# 특수 아이템 기본 드롭률 역산

## 구형 RNG 모델

구형 RNG는 `0..2³²-1`의 난수를 드롭률에 따라 줄어든 기준값으로 나눈 나머지로 당첨을 판정했다. 표기 추가 드롭률을 `D`(%), 아이템 상수를 `C`(0~1), 기본 드롭 임계값을 `T`(억 분율)라고 하면:

```text
B = floor(1,000,000,000 / (1 + D*C/100))
q = floor(2³² / B)
r = 2³² - q*B
```

이 문서의 특수 아이템은 모두 `T <= r`인 구간이므로 관측 확률은 `T*(q+1)/2³²`이다. 따라서 각 통계를 그 구간의 `q+1`로 나누어 기본 확률을 역산했다. [구형 RNG 모듈러 편향 설명](https://www.inven.co.kr/board/maple/5974/6456786)

## 솔 에르다 조각

[정리된 9개 케이스](https://www.inven.co.kr/board/maple/2304/47273)의 총합은 3,684,782마리, 3,471개다. 상수 50%와 각 케이스의 모듈러 배수를 적용한 유효 노출량은 8,176,113마리로, 합동 추정치는 `0.04245294%`다. 근사 95% 신뢰구간은 `0.04104~0.04386%`다.

`0.0425% = 425,000/10⁹`은 합동 추정치와 거의 같고 기존 통계의 다수 구간을 설명하는 정수 임계값이다. `0.04%`도 넓은 신뢰구간에서 완전히 배제할 수는 없지만, 합동 자료에 대한 적합도는 `0.0425%`가 더 높다.

## 코어 젬스톤

[총 359% 조건의 100만 마리 자료](https://www.inven.co.kr/board/maple/2304/19614)에서 `359%`는 기본 100% + 추가 259%다. 이를 추가 359%로 처리하면 모듈러 구간이 틀려진다. 추가 259%, 상수 50%의 `q+1`은 10이다.

- 1,723,459마리 / 1,119개: 역산 `0.027886%`
- 1,000,000마리 / 636개: 역산 `0.027316%`
- 두 표본 합계: 역산 `0.027676%`
- [약 800만 마리 추가 자료](https://www.inven.co.kr/board/maple/2304/11727): 조건 범위와 마릿수가 개략적이지만 역산값은 약 `0.02896%`

정확한 표본과 대형 개략 표본 사이의 중심값으로, `0.028% = 280,000/10⁹`을 사용한다.

## 심볼 교환권

기존 `0.00092%`는 위 2020년 자료의 **물방울석** 관측값 0.0042%에서 나온 값으로 보이며, 2023년에 추가된 어센틱심볼 교환권의 근거로 쓸 수 없다.

[드롭률 320%, 시간당 18,500마리 사냥 자료](https://www.inven.co.kr/board/maple/5974/3259888)의 평균은 코어 젬스톤 13개, 심볼 교환권 0.7개다. 두 아이템의 상수가 같으므로 비율을 이용하면 `0.028% * 0.7/13 = 0.001508%`다. [드롭률 40%/90% 10.5시간 자료](https://www.inven.co.kr/board/maple/5974/2149892)도 교환권이 매우 드문 아이템임을 보여주지만 마릿수가 없어 정밀 적합에는 사용하지 않았다.

현재는 가장 가까운 정수 임계값인 `0.0015% = 15,000/10⁹`을 사용한다. 다른 두 아이템보다 근거가 약하므로 충분한 마릿수와 획득량이 포함된 추가 자료가 필요하다.
